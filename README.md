# 📚 Product Filtering System

A modern and responsive **Product Filtering System** built with **HTML, CSS, and JavaScript**. This project allows users to browse a collection of books with **real-time search**, **genre filtering**, and **price sorting** through a clean and interactive interface.

## 🚀 Live Demo

🔗 **Demo:** https://sidraliaqat.github.io/Product-Filtering-System/

---

## ✨ Features

* 🔍 Real-time search by book title
* 🏷️ Filter books by genre
* 💰 Sort products by price (Low → High / High → Low)
* 📱 Fully responsive design
* 🎨 Modern glassmorphism user interface
* ⚡ Instant filtering without page reload
* 🖼️ Lazy-loaded images for better performance
* 📊 Dynamic product count

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Font Awesome
* GitHub Pages

## ⚙️ How It Works

1. Loads product data from `products.json`.
2. Displays all products dynamically.
3. Filters products as the user types.
4. Filters products by selected genre.
5. Sorts products by price.
6. Updates results instantly.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/product-filtering-system.git
```

Go to the project folder:

```bash
cd product-filtering-system
```

Open the project:

```bash
index.html
```

Or simply double-click **index.html**.
